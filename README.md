# Professional Portfolio - Miguel Motta

High-performance engineering portfolio showcasing expertise in Systems Engineering, Cloud Architecture, and AI Integration.

## 🚀 Tech Stack
- **Frontend**: React 18, TypeScript, Vite 6
- **Styling**: Tailwind CSS 4, shadcn/ui, Framer Motion
- **Tooling**: pnpm

## 🛠 Local Development
1. **Install dependencies**:
   ```bash
   pnpm install
   ```
2. **Run development server**:
   ```bash
   pnpm dev
   ```
3. **Build for production**:
   ```bash
   pnpm build
   ```

## 📁 Project Structure
- `src/app/data/profileData.ts`: Central source of truth for all portfolio content.
- `src/app/components/`: Page sections and layout logic.
- `src/app/components/ui/`: Reusable base UI components.
- `src/styles/`: Global CSS and theme configurations.
